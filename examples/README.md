# Examples

Some examples showing how to use `RESTArt-Mongo`.


## Dependencies

1. Install `RESTArt-Mongo`

    ```
    $ pip install RESTArt-Mongo
    ```

2. Download examples

    ```
    $ cd /tmp
    $ git clone https://github.com/RussellLuo/restart-mongo.git
    $ cd restart-mongo/examples
    ```
